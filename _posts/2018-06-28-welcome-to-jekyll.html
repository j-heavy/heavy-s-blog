Всем привет! Сегодня я наткнулся на один замечательный моноширный шрифт — Fira Code. Практически в каждом языке программирования есть большое количество составных символов. Так вот, данный шрифт упрощает чтение кода путём трансформации подобных символов в один.

Fira Code в деле

JavaScript

function getRandomInRange(min, max) {
	return Math.floor(Math.random() * (max - min + 1)) + min;
}

getRandomInRange(1, 10);

CSS

* {
	margin: 0;
  	padding: 0;
  	font-weight: normal;
}

body {
	font: 100%/1.5 Questrial, sans-serif;
	tab-size: 4;
	hyphens: auto;
}

Просто символы

=>
==
===
<=
->
<-
